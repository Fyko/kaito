# Introduction

### Features

- ✍ built in schema + body validation support
- ⚡ ultra quick, built on top of express
- 💪 robust
- 💨 extremely easy to use, built with decorators

## What is Kaito

Kaito is a minimalistic, HTTP framework for Node.js and TypeScript. It can also work in JavaScript environments, but
this may change in the future.

<Callout emoji="⚠">Kaito is not production ready, please use it with caution</Callout>
